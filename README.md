# Sinkhole Detector

An AI-powered sinkhole detection and analysis dashboard.

## Features

- Upload satellite and drone images
- Detect possible sinkholes
- Generate geological risk reports
- Display locations on a map
- Store scan history

## Technologies Used

- React
- TypeScript
- Express.js
- Google Gemini API
- Leaflet
